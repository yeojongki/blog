# 记录平时遇到的一些坑以及解决办法

目录结构如下：

```
├─audio-cant-play                               // IOS 下音频不能播放
├─canvas-crossorigin                            // 调用`canvas`的`drawImage`方法时，因为跨域问题而无法绘制
├─css3-android-landscape                        // 微信浏览器中安卓手机采用媒体查询判断横竖屏时有误
├─func-pass-arguments                           // JS函数中参数传递的方式
├─ios-date-transform                            // IOS下 new Date() 时间转换存在格式上的问题串中的连接符 `-`
└─parseInt                                      // 为什么['1', '2', '3'].map(parseInt)会返回[1, NaN, NaN]
```
